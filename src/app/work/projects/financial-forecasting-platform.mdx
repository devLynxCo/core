## Overview

At Forezynplan, I led the frontend development of a professional financial forecasting product (similar to Brixx). The project spanned both a scalable web application and cross-platform mobile apps for iOS and Android.

## Key Features

- **Scalable Web Application**: Built the main financial forecasting platform using Angular and TypeScript, focusing on data-heavy interfaces and complex user interactions.
- **Performance Optimization**: Improved load times by 40% using lazy loading strategies and efficient state management with NgRx.
- **Cross-Platform Mobile Apps**: Developed accompanying mobile apps using React Native and Expo, successfully achieving over 95% code sharing between iOS and Android.
- **High-Quality Mobile UX**: Enhanced the mobile user experience with smooth transitions using React Navigation and Reanimated 3, backed by Zustand and TanStack Query.

## Technologies Used

- **Angular & TypeScript**: Core stack for the complex web frontend.
- **NgRx**: For robust state management on the web application.
- **React Native (0.74+) & Expo**: For the mobile platform.
- **Jest**: For establishing and enforcing rigorous testing standards.

## Challenges and Learnings

The main challenge was handling complex financial calculations and large datasets on the client side without compromising performance. Introducing lazy loading and strict state management paradigms solved these issues. On the mobile side, ensuring native-like performance required careful architectural planning with React Native's Reanimated library.

## Outcome

The project resulted in a highly performant forecasting platform and mobile apps that maintained clean, reliable architectures. The emphasis on code sharing and structured testing practices ensured long-term maintainability for the product.
